# GPUPathTracer

A Wavefront path tracer written in C++/CUDA.

Current Features:
 - Next event estimation
 - Cosine weighted hemisphere sampling
 - Lambertian & purely specular materials
 - (Quad) MBVH acceleration structure with 32 byte nodes
 - Skybox

To Do:

 - More BSDFs
